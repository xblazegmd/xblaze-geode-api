# Xblaze's Geode API
A bunch of utility functions for my Geode mods/Patreon badges support

See the README on the project's repository for more info

## Patreon badges
To get your badge after you joined my [Patreon]() you need to go to [xblazegmd.vercel.app/patreon/link](https://xblazegmd.vercel.app/patreon/link) to link your Patron account to your Geometry Dash account

Your Account ID and Patreon account information will be stored in a database. If you wish to get that data removed go to [xblazegmd.vercel.app/patreon/remove](https://xblazegmd.vercel.app/patreon/remove)
